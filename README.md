# 🎨 Sibi Tattoo Studio Website

A modern, fully responsive website developed for **Sibi Tattoo Studio**, designed to showcase tattoo artistry, services, artist portfolio, pricing, and booking information with a premium user experience.

## 🌐 Live Demo

🔗 https://sibitattoostudio.netlify.app

---

# 📖 About the Project

Sibi Tattoo Studio is a professional tattoo studio website created to establish a strong online presence for the studio. The website provides visitors with an immersive experience by showcasing tattoo styles, artist expertise, studio environment, and appointment booking options.

The project focuses on delivering a modern UI/UX with smooth animations, responsive layouts, and an elegant dark aesthetic suitable for a premium tattoo brand.
